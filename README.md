# Project 2: Backend API Development (The Nervous System) 🧠

This project demonstrates a fully functional Node.js/Express Backend API integrated with a clean frontend interface. It is designed adhering to DecodeLabs guidelines, focusing on RESTful architecture, robust data validation, and correct semantic HTTP status codes.

## 🚀 Features & Architecture
- **RESTful Endpoints**: Proper naming conventions using nouns for resources (`GET /users`, `POST /users`).
- **The Gatekeeper Rule**: Implements syntactic and semantic input validation to reject bad or duplicate requests at the boundary.
- **CORS Enabled**: Configured to securely communicate with browser-based frontend clients.
- **Seamless Frontend**: Built-in AJAX (Fetch API) implementation for real-time user addition and listing without page reloads.

## 🛠️ Tech Stack
- **Backend**: Node.js, Express.js, CORS
- **Testing**: Thunder Client / Postman
- **Frontend**: HTML5, CSS3, JavaScript (ES6)
